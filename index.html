üêß Basic Linux Commands

pwd - Print working directory
Displays the current directory path.
ls - List directory contents
Shows files and folders in the current directory.
cd - Change directory
Used to move into another directory.
mkdir - Make directory
Creates a new folder/directory.
rm - Remove files/directories
Deletes files or folders (use with caution).
cp - Copy files/directories
Copies files from one location to another.
mv - Move or rename files
Moves files or changes their names.
touch - Create new file
Creates an empty file.
chmod - Change permissions
Modifies who can read/write/execute files.
chown - Change ownership
Changes the user/group that owns a file or folder.
grep - Search text
Finds patterns/words in text files.
ps - Process status
Displays currently running processes.
top - System resource monitor
Live view of system resource usage.
ifconfig/ip a - Network info
Shows IP address and interfaces.

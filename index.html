üìò Security+ Flashcards

Vulnerability: A weakness that can be exploited.
Exploit: Code/method to abuse a vulnerability.
Payload: Malicious code delivered through an exploit.
Brute Force: Trying many passwords until one works.
Dictionary Attack: Password guessing using wordlist.
Privilege Escalation: Gaining higher-level access.
Persistence: Maintaining access to a compromised system.
Lateral Movement: Moving through the network after gaining access.
Enumeration: Extracting detailed info about a target.
Reconnaissance: Information gathering before launching attacks.
Pivoting: Using a compromised machine to attack others.
Post-Exploitation: Actions taken after gaining access (like data theft).
TTPs: Tactics, Techniques, and Procedures used by attackers.
C2 (Command and Control): Server used to control compromised machines.
Red Team / Blue Team: Red = attackers, Blue = defenders.
Penetration Testing: Authorized testing for system vulnerabilities.
Social Engineering: Deceiving people to get access or information.
Zero-Day: An unknown/unpatched vulnerability.
RAT: Remote Access Trojan allowing full control of a system.
IDS/IPS: Intrusion Detection/Prevention Systems.
